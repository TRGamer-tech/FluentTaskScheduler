# Security Policy

## Supported Versions
| Version | Supported          |
| ------- | ------------------ |
| 1.5.x   | :white_check_mark: |
| < 1.5   | :x:                |

## Reporting a Vulnerability
Please report security vulnerabilities through public GitHub issues. 
We will try to acknowledge receipt of the vulnerability within 48 hours and work with you to resolve the issue.
